Q6) Create jobs_default Table with DEFAULT Values
Write an SQL query to create a table named jobs_default including columns job_id, job_title,
min_salary and max_salary.
Set default values:
- job_title default should be blank (“”)
- min_salary default should be 8000
- max_salary default should be NULL
mysql> create table job_default(job_id int,job_title varchar(20) default(' '),min_salary int default 8000,max_salary int default NULL);

Q7) Create departments Table (Composite Primary Key)
Create Department table with following structure:
Field Name Type Null Key
department_id DECIMAL(4,0) NO PRI
department_name VARCHAR(30) NO
manager_id DECIMAL(6,0) NO PRI
location_id DECIMAL(4,0) YES
Composite Primary Key: (department_id, manager_id)
mysql> create table department(dept_id decimal(4,0) NOT NULL,dept_name varchar(20) NOT NULL,manager_id decimal(6,0),location_id decimal(4,0),primary key(dept_id,manager_id));
